# Projeto Módulo 3 – Low Code/No Code/Vibecode

## 📌 Desafio Escolhido

Criar uma IA que transforma PDF em audio pensando na acessibilidade!
---

## 🖥️ Protótipo

- Prints das telas ou link para a aplicação criada.

- Breve explicação de como o protótipo funciona.

> Coloque os arquivos de imagem ou PDF na pasta `/docs`.

---

## ⚙️ Plataforma Utilizada

- Manus.ia ; Claude.ia ; ChatGPT.ia

- essas IA's foram esccolhidas por terem propositos diferentes, o chatgpt para gerar os textos, a claude para refinar o codigo e o Manus para gerar o codigo bruto;

---

## ✅ Vantagens Identificadas


1. Protótipo rápido

2. Integração simples

3. Automação de processos

---

## ⚠️ Limitações Encontradas


1. Customização limitada

2. Dependência da plataforma

3. Risco de lock-in tecnológico

---

## 📚 Reflexão Crítica

As limitações encontradas pelo grupo foi o uso limitado em cada plataforma por necessitar melhor o plano contratado ou na quantidade de criações
